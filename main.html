Koteky

All your social content in one place

One App To Rule All

One aplication to work with all your social networks. Any social network can be added as an extension to this application, incrising your power to manage your hability to follow up with the content that is normally disperse in between different applications. Each of the the extensions can expose the diferences that makes each of the social networks unique so you dont lose on the small things. Post into all network without content switching.

Cross-Platform

Run this application under any operating system. Same look an feel in all of them.

Extensible

Create your own plugins to extend the application through npm plugins and plain javascript

Open Source

all the code can be found in github
